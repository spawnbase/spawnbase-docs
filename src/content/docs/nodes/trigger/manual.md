## Usage

Click **Run** in the agent editor to execute the workflow immediately. Manual runs use the same execution pipeline as scheduled or webhook-triggered runs.

## When to Use

- Testing and debugging workflows during development
- One-off tasks that don't need automation
- Agents that are triggered by humans rather than events
